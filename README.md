# 📊 Hadoop WordCount Project (CLI + UI)

## 📌 Overview
This project implements the Word Count problem using Hadoop MapReduce in Java.  
It also includes a **Streamlit-based interactive UI** for uploading files and visualizing results.

---

## ⚙️ Technologies Used
- Java 8
- Hadoop 3.3.6
- Python (Streamlit)
- WSL (Ubuntu)

---

## 🚀 Features
- ✔ Case-insensitive word counting  
- ✔ Punctuation handling  
- ✔ Hadoop MapReduce processing  
- ✔ One-command CLI execution  
- ✔ Interactive Web UI  
- ✔ Top words analysis  
- ✔ Visualization (bar chart)  

---

## 📂 Project Structure
src/ → Java source code
input/ → Input text files
run.sh → CLI execution script
app.py → Streamlit UI
requirements.txt → Python dependencies
